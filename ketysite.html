Sobre o Bot Kety

O Bot Kety é uma ferramenta poderosa criada para facilitar a administração de servidores Discord. Ele oferece uma série de funcionalidades de moderação e automação para garantir que seu servidor funcione de maneira eficiente e sem problemas. Com Kety, você pode automatizar tarefas, gerenciar membros, configurar permissões, e muito mais, tudo de forma simples e rápida.

Benefícios da Bot Kety

  • Moderação automática: Filtros de conteúdo, banimento e mutação automáticos, e muito mais para garantir que seu servidor seja sempre seguro.
  • Automação de tarefas: Agende mensagens, comandos personalizados e ações automáticas para tornar o gerenciamento do servidor mais eficiente.
  • Suporte para múltiplos idiomas: O Bot Kety pode ser usado em diferentes idiomas, tornando-o acessível para comunidades globais.
  • Fácil de configurar: Interface simples para configurar todas as funções sem complicações.

Como Usar o Bot Kety

Usar o Bot Kety é simples e direto. Siga as etapas abaixo para começar:

  1. Adicione o bot ao seu servidor Discord usando o link de convite disponível.
  2. Use o comando -runask para fazer perguntas ao Kety.
  3. Use o comando -runkick para expulsar membros.
  4. Use o comando -runask para dar ban no usuário.
  5. Use o comando -run_removerole para remover o cargo de um membro, impedindo que ele escreva em qualquer lugar do servidor.

Com essas simples etapas, você terá o controle total do seu servidor Discord com o poder do Bot Kety!
